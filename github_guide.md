## USUL 1: GitHub Web Orqali (Eng Oson)

### Qadam 1: Repository Yaratish
1. GitHub.com ga kiring
2. O'ng yuqorida "+" belgisini bosing
3. "New repository" ni tanlang
4. To'ldiring:
   - Repository name: `plan-tracker`
   - Description: `Professional Planner & Finance Tracker`
   - Public ‚úÖ (belgilang)
   - Initialize this repository with a README ‚ùå (belgilamang!)
5. "Create repository" tugmasini bosing

### Qadam 2: Fayllarni Yuklash
1. Yangi ochilgan sahifada "uploading an existing file" linkini bosing
2. Plan Tracker papkasidan BARCHA fayllarni sudrab tashlang:
   ‚úÖ index.html
   ‚úÖ index.css
   ‚úÖ main.js
   ‚úÖ manifest.json
   ‚úÖ sw.js
   ‚úÖ README.md
   ‚úÖ LICENSE
   ‚úÖ .gitignore
3. Commit message: "Initial commit"
4. "Commit changes" tugmasini bosing

### Qadam 3: GitHub Pages Yoqish
1. Yuqorida "Settings" (sozlamalar) ga kiring
2. Chap menyudan "Pages" ni toping
3. "Source" qismida:
   - Branch: `main` ni tanlang
   - Folder: `/ (root)` ni tanlang
4. "Save" tugmasini bosing
5. 1-2 daqiqa kuting
6. Sahifani yangilang (F5)
7. Link paydo bo'ladi: https://[username].github.io/plan-tracker
